A new plugin...

28th February 2018 at 11:39am

I learned how to create a plugin today, using Tinka - the easy plugin packer. The plugin is called DesignWriteStudio - showNotes:

This plugin supports the use of the <<show>> macro to share notes or comments. Visibility of the notes is controlled by the show dox checkbox in the top left menu. This plugin is from the DesignWriteStudio community.
If the So you should be able to drag/drop from the plugins tab on the $:/ControlPanel to your wiki. Let me know if it works (or not) in the GoogleGroup.
