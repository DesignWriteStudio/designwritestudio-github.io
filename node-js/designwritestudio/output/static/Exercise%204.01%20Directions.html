(these directions were generated in the Workshop: Annotating Sources)

  1. Create new wiki/tiddler
  2. Identify and download text of two articles that looks interesting: one focused on hypertext and one focused on annotation
  3. Ingest each article into their own tiddlers
  4. Use excise tool to split articles into paragraphs or sections
  5. Use some of these techniques (or others that you develop) to annotate the articles:
    1. Add tags to paragraph/section tiddlers
    2. [[bracket keywords]] to make new tiddlers
    3. Use new here to create annotations
