Exercise 3.03 Directions

14th February 2018 at 3:37pm
  1. Use google sheets to ingest (import) a table from wikipedia, using the =importhtml() command
  2. I suggest you insert a row above the wikipedia table and use it to create short, easy-to-understand fieldnames for each column
  3. Download your spreadsheet as an xlsx
  4. Create a new tiddlywiki (or build on to an existing tiddlywiki file)
  5. Navigate to the tab, select Get More Plugins and search for/install two plugins:
    1. xlsx utilities
    2. jszip
  6. Save and refresh your wiki
  7. Click on the tab.
    1. Create a new workbook (any name is fine)
    2. Create a new sheet (most likely, your sheet name will be Sheet1)
    3. Create a new row.
    4. Add a new field for each column in the spreadsheet. It is easiest to name your fields with the same names as the spreadsheet columns. Be sure to exactly replicate the column name in the appropriate space in the form
    5. Add a new field called title and set it to equal the contents of a column that has unique values (such as Mountain in Wikipedia: List_of_highest_mountains_on_Earth )
    6. Add a new field called tags and set it to a constant value that describes your objects (i.e. mountains, cars, etc).
    7. Save your wiki, and reload
  8. Select your new workbook as the Current Import Specification in the tab
  9. Use the import tool
    and import your downloaded xlsx file.
  10. Hope for the best!
  11. If it works
    1. Create a template to display contents of your tagged tiddlers (i.e. "mountains") with at least two fields.
